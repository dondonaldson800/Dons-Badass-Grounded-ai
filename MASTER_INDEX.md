## 🆘 TROUBLESHOOTING

### Cannot find ZIP files
**Location:** `/app/app-archives/`
**Check:** `ls -lh /app/app-archives/`

### ZIP extraction fails
```bash
unzip -l file.zip  # Test ZIP integrity
unzip file.zip     # Extract
```

### Dependencies fail to install
```bash
cd expo-app[X]
rm -rf node_modules yarn.lock
yarn install
```

### EAS build fails
```bash
# Check credentials
eas whoami

# Reconfigure
eas build:configure

# Try again
eas build --platform android --profile production
```
