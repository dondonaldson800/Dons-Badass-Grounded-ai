# Photo Studio

Photography portfolio and editing

## Bundle ID
- Android: `com.empire.photography`
- iOS: `com.empire.photography`

## Theme
- Primary: #0D1B2A
- Secondary: #1B263B

## Screens
1. HomeScreen
2. GalleryScreen
3. EditScreen
4. ShareScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
