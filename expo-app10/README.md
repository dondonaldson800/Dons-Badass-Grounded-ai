# Pet Care

Track pet health and appointments

## Bundle ID
- Android: `com.empire.petcare`
- iOS: `com.empire.petcare`

## Theme
- Primary: #F72585
- Secondary: #B5179E

## Screens
1. HomeScreen
2. PetsScreen
3. HealthScreen
4. RemindersScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
