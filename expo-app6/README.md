# Recipe Hub

Discover recipes and plan weekly meals

## Bundle ID
- Android: `com.empire.recipes`
- iOS: `com.empire.recipes`

## Theme
- Primary: #2A9D8F
- Secondary: #1B7A6E

## Screens
1. HomeScreen
2. RecipesScreen
3. Meal PlanScreen
4. GroceryScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
