# Sports News

Live scores and sports updates

## Bundle ID
- Android: `com.empire.sports`
- iOS: `com.empire.sports`

## Theme
- Primary: #06A77D
- Secondary: #048B64

## Screens
1. HomeScreen
2. NewsScreen
3. ScoresScreen
4. FavoritesScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
