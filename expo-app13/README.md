# Home Garden

Garden planning and plant care

## Bundle ID
- Android: `com.empire.homegarden`
- iOS: `com.empire.homegarden`

## Theme
- Primary: #6A4C93
- Secondary: #563D7C

## Screens
1. HomeScreen
2. PlantsScreen
3. CalendarScreen
4. TipsScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
