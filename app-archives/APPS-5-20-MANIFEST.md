## 🚀 SETUP INSTRUCTIONS

### 1. Extract Archive
```bash
unzip app5-fit-empire.zip
cd fit-empire
```

### 2. Install Dependencies
```bash
yarn install
# or
npm install
```

### 3. Run Locally
```bash
npx expo start
```

### 4. Initialize Git & Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit: [App Name]"
git remote add origin https://github.com/YOUR_ORG/[repo-name].git
git push -u origin main
```

### 5. Configure EAS
```bash
eas login
eas build:configure
```

### 6. Build for Production
```bash
# Android APK (for testing/side-loading)
eas build --platform android --profile preview

# Android AAB (for Google Play)
eas build --platform android --profile production
```

## 📝 CUSTOMIZATION GUIDE

### Change App Name
Edit `app.json`:
```json
{
  "expo": {
    "name": "Your Custom Name",
    "slug": "your-custom-slug"
  }
}
```

### Change Bundle ID
Edit `app.json`:
```json
{
  "expo": {
    "android": {
      "package": "com.yourcompany.appname"
    },
    "ios": {
      "bundleIdentifier": "com.yourcompany.appname"
    }
  }
}
```

### Change Theme Colors
Edit `App.js`:
```javascript
export const AppTheme = {
  colors: {
    primary: '#YOUR_COLOR',
    secondary: '#YOUR_SECONDARY',
    // ...
  },
};
```

### Add More Screens
1. Create new screen file in `screens/`
2. Import in `App.js`
3. Add to `Tab.Navigator`

## 🆘 SUPPORT & TROUBLESHOOTING

### Common Issues

**"Module not found" error:**
```bash
rm -rf node_modules
yarn install
```

**"Unable to resolve module" error:**
```bash
npx expo start -c  # Clear cache
```

**EAS build fails:**
- Verify bundle ID is unique
- Check `eas.json` configuration
- Ensure you're logged in: `eas whoami`

**Navigation not working:**
- Verify all screens are imported in `App.js`
- Check screen names match between import and `Tab.Screen`
