# Style Guide

Fashion inspiration and wardrobe management

## Bundle ID
- Android: `com.empire.fashion`
- iOS: `com.empire.fashion`

## Theme
- Primary: #CB997E
- Secondary: #B07D62

## Screens
1. HomeScreen
2. OutfitsScreen
3. WardrobeScreen
4. TrendsScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
