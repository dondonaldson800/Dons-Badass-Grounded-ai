## 📱 HOW TO GET YOUR APK (3 Easy Steps)

### STEP 1: Push Code to GitHub

1. **Use Emergent's "Save to GitHub" button**
   - Click "Save to GitHub" in Emergent interface
   - Connect your GitHub account
   - All code pushed automatically!

OR

2. **Manual Git push** (if you have the code locally):
```bash
git init
git add .
git commit -m "D's Empire complete"
git remote add origin https://github.com/YOUR_USERNAME/dempire.git
git push -u origin main
```

---

### STEP 2: GitHub Builds APK Automatically

Once code is on GitHub:
1. Go to your repo → **Actions** tab
2. You'll see "Build Android APK" workflow running
3. Wait **10-15 minutes** (first build takes longer)
4. Build completes automatically! ✅

---

### STEP 3: Download Your APK

**Option A - From Artifacts:**
1. Click on the completed workflow
2. Scroll down to **Artifacts**
3. Download `dempire-debug-apk.zip`
4. Extract → You have `app-debug.apk`!

**Option B - From Releases:**
1. Go to **Releases** tab in your repo
2. Find latest release (v1.0.X)
3. Download `app-debug.apk` directly
4. Install on Android phone!
