# Music Player

Music streaming and playlist management

## Bundle ID
- Android: `com.empire.music`
- iOS: `com.empire.music`

## Theme
- Primary: #D90429
- Secondary: #9D0208

## Screens
1. HomeScreen
2. LibraryScreen
3. PlaylistsScreen
4. PlayerScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
