# Real Estate Pro

Browse properties and calculate mortgages

## Bundle ID
- Android: `com.empire.realestate`
- iOS: `com.empire.realestate`

## Theme
- Primary: #0096C7
- Secondary: #0077B6

## Screens
1. HomeScreen
2. ListingsScreen
3. FavoritesScreen
4. CalculatorScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
