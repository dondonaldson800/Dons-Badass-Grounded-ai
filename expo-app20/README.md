# Zen Wellness

Meditation, mindfulness, and wellness tracking

## Bundle ID
- Android: `com.empire.wellness`
- iOS: `com.empire.wellness`

## Theme
- Primary: #9D4EDD
- Secondary: #7B2CBF

## Screens
1. HomeScreen
2. MeditateScreen
3. JournalScreen
4. ProgressScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
