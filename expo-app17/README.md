# Book Tracker

Track reading progress and reviews

## Bundle ID
- Android: `com.empire.books`
- iOS: `com.empire.books`

## Theme
- Primary: #6A040F
- Secondary: #370617

## Screens
1. HomeScreen
2. LibraryScreen
3. ReadingScreen
4. StatsScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
