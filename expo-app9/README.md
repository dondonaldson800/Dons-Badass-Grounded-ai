# Travel Guide

Plan trips and explore destinations

## Bundle ID
- Android: `com.empire.travel`
- iOS: `com.empire.travel`

## Theme
- Primary: #F77F00
- Secondary: #DC2F02

## Screens
1. HomeScreen
2. DestinationsScreen
3. ItineraryScreen
4. TipsScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
