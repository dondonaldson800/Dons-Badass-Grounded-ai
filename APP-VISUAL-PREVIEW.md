## ⚡ INTERACTIONS

**Home Screen:**
- Tap "Support the Cause" → Navigate to Donate screen
- Tap "Volunteer" → Navigate to Volunteer screen
- Tap "Share Our Mission" → Opens native share sheet

**Impact Screen:**
- Scroll to see monthly trends
- View transparency breakdown
- Progress bars animate on load

**Donate Screen:**
- Tap donation amount card → Card highlights in orange
- Selected amount → Donate button appears
- Tap Donate → Shows payment integration placeholder

**Volunteer Screen:**
- Tap "Sign Me Up" → Confirmation dialog appears
- Confirm → Updates volunteer count
- Full event → Shows "Event Full" alert

**To see this live:**
1. Download `/app/core-four-apps.zip`
2. Extract and navigate to `expo-nonprofit`
3. Run `yarn install`
4. Run `npx expo start`
5. Scan QR code with Expo Go app on your phone

- Smooth bottom tab navigation
- Realistic nonprofit dashboard
- Interactive volunteer sign-ups
- Professional donation flow
- Social sharing integration
