# Auto Care

Vehicle maintenance tracking and reminders

## Bundle ID
- Android: `com.empire.automotive`
- iOS: `com.empire.automotive`

## Theme
- Primary: #1D3557
- Secondary: #14213D

## Screens
1. HomeScreen
2. VehiclesScreen
3. MaintenanceScreen
4. LogsScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
