# Budget Master

Manage finances and track spending

## Bundle ID
- Android: `com.empire.finance`
- iOS: `com.empire.finance`

## Theme
- Primary: #7209B7
- Secondary: #560BAD

## Screens
1. HomeScreen
2. ExpensesScreen
3. BudgetScreen
4. ReportsScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
