# Fit Empire

Track workouts, calories, and fitness goals

## Bundle ID
- Android: `com.empire.fitness`
- iOS: `com.empire.fitness`

## Theme
- Primary: #E63946
- Secondary: #A4161A

## Screens
1. HomeScreen
2. WorkoutsScreen
3. ProgressScreen
4. ProfileScreen

## Setup

```bash
yarn install
expo start
```

## Build

```bash
eas build --platform android --profile production
```
